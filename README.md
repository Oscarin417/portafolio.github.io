# Portafolio - Oscar Ramirez

Portafolio web personal showcasing proyectos y habilidades como Desarrollador de Software.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS
- **UI Components:** Shadcn/ui

## Secciones

- **Hero** - Presentación con animación de cursor
- **Sobre mí** - Descripción profesional
- **Proyectos** - 5 proyectos con tecnologías usadas
- **Habilidades** - Tech stack organizado por categorías
- **Contacto** - Links a LinkedIn, GitHub y Email

## Preview

Tema dark mode con estilo IDE moderno. Colores cyan/purple con efectos glow.

## Setup Local

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Abrir en navegador
http://localhost:3000
```

## Deploy

Desplegar en [Vercel](https://vercel.com) (recomendado) o cualquier hosting compatible con Next.js.
